Java基础之带标签的break和continue

Java基础之带标签的break和continue

七月 07, 2019 阅读数(请刷新)
  • 学习

标签是指后面跟一个冒号的标识符，例如：“lable：”

对Java来说唯一用到标签的地方是在循环语句之前，而在循环之前设置标签的为唯一理由是：希望在其中嵌套另一个循环。由于break和continue关键字通常只中断当前循环，若随同标签使用，他们就会中断到存在标签的地方。
